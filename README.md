# driving-light-style

自動車走行時に見やすい地図スタイル・ライトモード (Driving-friendly map style - Light mode)

明るい背景に高コントラストの道路表示で、昼間の運転中にちらっと見るだけで道路情報が把握できるスタイルです。

## プレビュー

https://geoloniamaps.github.io/driving-light-style/

## 使い方

```
https://geoloniamaps.github.io/driving-light-style/style.json
```

## 開発

```bash
npm install
npm start
```

## ビルド

```bash
npm run build
```

`docs/style.json` が生成されます。GitHub Pages で配信されます。
